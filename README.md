“Wun geoi” (玩具) means toy in Cantonese.

When I asked my 2-year-old son to describe what matters most to him right now, he simply repeated: "wun geoi, wun geoi."

Inspired by that simplicity and focus, `wungeoi` is also the name of this core time-series forecasting toolbox — designed to be a foundational building block (a "toy") for powering a variety of downstream tasks.
